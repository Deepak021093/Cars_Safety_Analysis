Cars_Safety_Analysis
 
Aim:
To analyse the dataset containing details of NCT vehicle inspection, and trying to find out which is the safest car maker from the given details.

Note: 

Please change the dataset name to the appropriate path in the R file, to be able to run the codes successfully. (line 32 and 346)

The folder named "Codes"ù contain the R codes to be able to generate the html outputs, and the folder named "Datasets"ù contains the datasets which were used for this analysis.
